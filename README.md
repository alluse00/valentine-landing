# San Valentin Landing

Base visual para una propuesta del Dia de San Valentin. Muestra el titulo "San Valentin" sobre un fondo visual con botones listos para enlazar a futuras secciones.

## Uso local
1. Abre index.html directamente en tu navegador.
2. Ajusta textos o estilos en index.html y styles/main.css.

## Compartir por enlace
- Puedes subir esta carpeta a cualquier hosting estatico (GitHub Pages, Netlify, Vercel, etc.).
- En GitHub Pages: sube los archivos a un repositorio y habilita Pages apuntando a la rama principal; el enlace publico abrira index.html en cualquier dispositivo.

## Estructura
- index.html: punto de entrada.
- styles/main.css: estilos base y hero.
- assets/: carpeta para imagenes o fuentes adicionales (crea el archivo necesario cuando lo uses).
